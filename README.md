# Fake-News-Detection-Using-BERT

This project focuses on classifying news articles as real or fake using a transformer-based NLP model. The objective is to leverage transfer learning through BERT to improve text classification performance.

# Problem Statement

The spread of misinformation through online platforms has become a significant challenge. This project aims to automatically classify news articles as real or fake using Natural Language Processing techniques.

# Dataset

- Fake News Dataset
- Binary Classification Task (Fake / Real)

**Source - Kaggle**

# Approach

- Text Preprocessing
- Tokenization using BERT Tokenizer
- Dataset Preparation
- Fine-Tuning Pretrained BERT
- Model Evaluation

# Models Used

- BERT Base Uncased

# Results

The model was fine-tuned on labeled news articles and evaluated on unseen test data for fake news classification.
